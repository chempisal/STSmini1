បញ្ចូលតំណភ្ជាប់ Google Apps Script Web App URL ដើម្បីធ្វើសមកាលកម្មទិន្នន័យទៅកាន់ Google Sheets របស់អ្នកដោយស្វ័យប្រវត្តិ។

💡 របៀបដំឡើង៖
  1. បង្កើត Google Sheet ថ្មីមួយ និងចូលទៅកាន់ Extensions ➔ Apps Script
  2. ចម្លងកូដ Apps Script ពីឯកសារគម្រោង ឬ ចុចទីនេះដើម្បីចម្លងកូដគំរូ
  3. ចុចលើប៊ូតុង Deploy ➔ New Deployment
  4. ជ្រើសរើសប្រភេទជា Web App, កំណត់ការចូលប្រើប្រាស់ទៅជា Anyone រួចចុច Deploy
  5. ចម្លងតំណភ្ជាប់ (Web App URL) មកដាក់ក្នុងប្រអប់ខាងលើនេះជាការស្រេច។
